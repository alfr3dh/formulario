This plugin generates QR codes for URLs. It will automatically generate a QR code for the current page URL.

== Description ==

[QR code Composer](https://sharabindu.com/plugins/qr-code-composer/) is a plugin for generating QR codes for URLs. It will automatically generate a QR code for the current page URL.

QR codes are automatically displayed after the content. It acts as the current page URL content.

It will start working as soon as the plugin is activated. You can turn off QR codes for specific pages or specific post types if you don't want them.

Each QR code is unique and will be generated when you publish a post, page,  product, or custom post.

**Free Version Features**


* Generating dynamic QR code based on current page URL

* The QR code is automatically displayed after the content

* Generate vCard QR Code [qrc_vcard_single]

* Generate vCard QR Code for WordPress Users

* Generate vCard QR Code for WooCommerce Customer

* Generate vCard QR Code for WooCommerce Customer Billing Address and Shipping Address

* QR code size can be customized easily


* Alignment can be customized

* It is possible to remove QR codes for specific pages, posts, products, or custom posts

* Easily remove QR codes for specific post types

* QR code can be downloaded from the backend and frontend

* Customizing the download button and its typography

* Customizing the title (QR code) of a single product page tab



**Automatically display the Bottom of the content:​** QR codes are automatically displayed after the content. It acts as the current page URL content. That is, for the QR code of the current page URL, you can display the QR code below the content. It will start working as soon as the plugin is activated. You can turn off these features if you don't want them. You get many options to turn off these features, such as by post type or by page, and by meta box. See the [documentation](https://qrcode-composer.dipashi.com/docs/current-page-settings/) for details. These features available for both free and premium versions



** **Premium Version Features** **

**Colorful QR code: Users can add color to the QR code to decorate it. You can easily make QR codes attractive by adding dot color and background color of QR codes.


**Logo Image Upload:** User can add the company logo in the center of the QR code, which can be used in company brand QR codes. It will serve as an element of your company's advertising and marketing. QR codes will create separate entities

**QR code for Custom text/custom link:** Users can create QR codes based on custom text or custom links. They can generate multiple QR codes in these features using the shortcode attributes. If they are users of the Elementor, they can do this too.

**QR Code for Google Maps:** Users can create a QR code location for the address. This way, the company location can be found simply by scanning the code. Customers no longer need to manually enter an address, which significantly reduces the possibility of mistakes or getting lost.


**WhatsApp QR code:** A WhatsApp QR code is a Quick Response code that enables users to add their friends and family’s contact information on WhatsApp.

**WiFi QR Code:**Scan QR Code to Connect To Any WiFi Network Instantly. A WiFi QR Code is a solution that lets you create QR Codes to connect to the internet. Users can scan the assigned WiFi QR Code using their smartphone’s camera or third-party QR Code scanner apps without the need to type in the password. 


**vCard QR Code Generator**

* What is a vCard QR Code?
A vCard QR Code works as a virtual contact card, allowing you to share your contact details seamlessly on mobile devices. The QR code includes your contact information, like name, phone number, email address, etc. Users can scan the vCard QR code to get their information and save your contact


* Generate vCard QR Codes via Metabox: This is a great way to generate multiple vCard QR codes. It works on the basis of the post type. See this [link](https://qrcode-composer.dipashi.com/docs/vcard-metabox/), for how to generate multiple vCard QR codes with Metabox.

* Bulk vCard QR code generation by shortcode and its attributes: vCard QR code you need to use a shortcode with an attribute. Change the attribute and create a bulk vCard as per your requirement. [qrc_vcard_single name=" " company=" " subtitle=" " mobile=" " phone=" " email=" " website=" " address =" " note ="" logo="" color="#311cb5" background="#fff" size="250"]


**Display Method for QR codes**


**Shortcode Supported:** You can Generate and display QR codes using the plugin's built-in shortcodes. Each shortcode has attributes. So can Generate multiple QR codes for different components

* QR code for Current Page URL: [qrc_code_composer]

* QR code for Custom text or custom link or Number: [qr_link_composer link=""]

* Whatsapp QR code: [qr_whatsapp_composer Whatsapp =""]

* Wifi QR code: [qr_wifi_composer wifi_name=" " wifi_type=" " wifi_password=" "]

* QR code for Google Map: [qr_maps_composer latitude=" " longitude=" " query=" "]

* vCard QR code: [qrc_vcard_single name=" " company=" " subtitle=" " mobile=" " phone=" " email=" " website=" " address =" " note =""]

* Event QR code: [qrc_event summary="" description="" location="" startdate="" starttime="" enddate="" endtime=""]

**===== Integration ======**

**BBPress Plugin:** QR Code Composer Pro plugin, generates QR codes for bbPress users. bbPress users can generate QR codes for their own user profile links, and they can generate vCard QR codes based on user information, such as name, nickname, email, etc. [View demo](https://qrcode-composer.dipashi.com/forums/users/dipashi/)


**BuddyPress Plugin:** QR Code Composer Pro plugin generates Qr codes for BuddyPress users. BuddyPress users can generate QR codes for their own users' profile links, and they can generate vCard QR codes based on user information, such as name, nickname, email, etc. [View demo](https://qrcode-composer.dipashi.com/members/dipashi/profile/qr-code/)



**Dokan Plugin:** QR Code Composer Pro plugin generates QR codes for Dokan vendors. Dokan users can generate QR codes for their vendor's profile links, and they can generate vCard QR codes based on the vendor's information, such as name, address, email, etc. [View demo](https://qrcode-composer.dipashi.com/store/alexa-expreee/)

== Installation ==

Go to the Plugins Menu in WordPress and select Plugins > Add new
Search for “ QR Code Composer”
Click “Install” and “Activate”.
